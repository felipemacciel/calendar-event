## Como rodar este projeto?
```bash

# Clone este repositório
$ git clone [linkrepo](https://github.com/felipemacciel/calendar-event)

# Acesse a pasta do projeto no seu terminal
$ git clone [linkrepo](https://github.com/felipemacciel/calendar-event)

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run start
```
